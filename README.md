## Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```
   Opens automatically at `http://localhost:3000`

3. **Build for production**
   ```bash
   npm run build
   ```

## How to Use

1. **Add Application** - Click "+ Add Application" button
2. **Fill the form** - Company, position, location, salary, status, notes
3. **Search & Filter** - Use search box and status buttons to filter
4. **Edit** - Click Edit button on any card to update details
5. **Delete** - Click Delete button to remove (with confirmation)
6. **Data persistence** - All data is saved automatically to browser

## Data Storage

- Data is stored in browser's **localStorage**
- No server or account needed
- Persists even after closing browser
- To clear all data: Open DevTools > Application > localStorage > Clear
